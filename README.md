# library-database
Курсовая работа по базам данных. База данных для автоматизации библиотеки.


## СУБД
PostgreSQL


## Таблицы

В базе данных реализовано 6 таблиц:

- **libraries** — библиотеки (название, адрес, телефон)
- **books** — книги (автор, название, издательство, год издания, количество)
- **genres** — жанры книг
- **readers** — читатели библиотеки
- **librarians** — сотрудники библиотек
- **loans** — выдача книг читателям

## Ограничения

- **PRIMARY KEY**
- **FOREIGN KEY**
- **UNIQUE**
- **CHECK**
- **DEFAULT**

## Индексы

- `idx_books_title`
- `idx_readers_name`
- `idx_loans_reader`

## Представления (VIEW)

1. **available_books** — доступные книги (количество > 0)
2. **borrowed_books** — список выданных книг
3. **popular_books** — популярные книги (использует GROUP BY и HAVING)

## Триггеры

- **trg_decrease_books** — уменьшает количество книг при выдаче
- **trg_return_book** — увеличивает количество книг при возврате
